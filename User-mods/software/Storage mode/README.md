### Storage mode

#### Реализация режима хранения макросами Klipper

С помощью переменной storage_after_timer [0/1] в макросе DRY_U1 задается сценарий по умолчанию, после окончания таймера сушилка либо отключается [0], либо переходит в режим хранения [1]. Макросом STORAGE_AFTER_TIMER_U1 можно быстро изменить значение этой переменной "на лету", сохраняется до перезагрузки клиппера.

Если переменная storage_after_timer = 1, после окончания таймера заданного в пресете (PETG_U1, ABS_U1 и т.п.) сушилка переходит в режим хранения: нагрев камеры отключается, серва всегда закрыта, индикатор влажности становится зеленым, непрерывно оценивается влажность в камере. Когда влажность становится выше влажности заданной в пресете то включается нагрев камеры пока влажность не уменьшится. Предусмотрен гистерезис по влажности (переменная hyst_humidity в макросе DRY_U1) для уменьшения частоты включения нагревания. Температура нагрева камеры равна температуре из пресета. Режим хранения бесконечен, остановить его можно макросом OFF_U1. Режим хранения можно включить принудительно макросом STORAGE_U1.

#### Логика индикации

1. Индикатор нагревателя. Нагреватель включен - пурпурный. Нагреватель выключен - слабый белый.
2. Индикатор вентилятора. Вентилятор включен - голубой. Вентилятор выключен - слабый белый.
3. Индикатор влажности. От 5 до 30 процентов - градиент от оранжевого до синего. В режиме хранения - зеленый.
4. Индикатор температуры. От 25 градусов до температуры из пресета - градиент от синего до красного.
При выключенной сушилке все индикаторы горят слабым белым.
Переменными в макросе DRY_U1 можно задать яркость индикаторов.

#### Прочее
Переменной telegram в макросе DRY_U1 можно включить/отключить сообщения телеграм боту. Макросом TELEGRAM_U1 можно изменить значение этой переменной "на лету", сохраняется до перезагрузки клиппера.

Макрос STATUS_U1 выводит: статус сушилки, температуру в камере, влажность, состояние нагревателя и переменной storage_after_timer.

#### Установка

Скопировать файл U1.cfg в папку с конфигами iDryer.
