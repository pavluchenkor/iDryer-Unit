### Storage mode

####Реализация режима хранения макросами Klipper

После окончания таймера заданного в пресете (PETG_U1, ABS_U1 и т.п.) сушилка переходит в режим хранения: нагрев камеры отключается, серва всегда закрыта, индикатор влажности становится зеленым, непрерывно оценивается влажность в камере. Когда влажность становится выше влажности заданной в пресете то включается нагрев камеры пока влажность не уменьшится. Предусмотрен гистерезис по влажности (переменная hyst_humidity в макросе DRY_U1) для уменьшения частоты включения нагревания. Температура нагрева камеры равна температуре из пресета. Режим хранения бесконечен, остановить его можно макросом OFF_U1. Режим хранения можно включить принудительно макросом STORAGE_U1.

####Переработана логика индикации

1. Индикатор нагревателя. Нагреватель включен - пурпурный. Нагреватель выключен - слабый белый.
2. Индикатор вентилятора. Вентилятор включен - голубой. Вентилятор выключен - слабый белый.
3. Индикатор влажности. От 5 до 30 процентов - градиент от оранжевого до синего. В режиме хранения - зеленый.
4. Индикатор температуры. От 25 градусов до температуры из пресета - градиент от синего до красного.
При выключенной сушилке все индикаторы горят слабым белыс.
Перемнными в макросе DRY_U1 можно задать яркость индикаторов.

####Прочее
Переменной telegram в макросе DRY_U1 можно включить/отключить сообщения телеграм боту.

####Установка

Скопировать файл U1.cfg в папку с конфигами iDryer.
