### Storage mode

#### Реализация режима хранения макросами Klipper

Логика следующая: после окончания таймера сушки заданного в пресете (PETG_U1, ABS_U1 и т.п.) включается режим хранения: заслонка проветривания всегда закрыта, все индикаторы горят зеленым (чтобы понимать, что сушилка находится в режиме хранения), дальше непрерывно оценивается влажность в камере. Если влажность становится выше влажности заданной в пресете сушки + delta_humidity (гистерезис) то включается нагрев камеры пока влажность не упадет до влажности из пресета. Температура нагрева равна температуре из пресета. Режим хранения бесконечен, остановить его можно макросом OFF_U1. Режим хранения можно включить принудительно макросом STORAGE_U1, нужно будет задать температуру и влажность. В макросе DRY_U1 настраивается дельта влажности: delta_humidity.

#### Переработана логика индикации

1. Индикатор нагревателя. Нагреватель включен - яркий красный. Нагреватель выключен - слабый белый.
2. Индикатор вентилятора. Вентилятор включен - яркий зеленый. Вентилятор выключен - слабый белый.
3. Индикатор влажности. Синий. По мере уменьшения влажности яркость снижается. Меньше 10% - становится зеленым.
4. Индикатор температуры. Красный. Меняет яркость в завсисимости от температуры.
При выключенной сушилка все индикаторы горят слабым синим. В режиме хранения - слабый зеленый.

#### Установка


Скопировать файл U1.cfg в папку с конфигами iDryer.
